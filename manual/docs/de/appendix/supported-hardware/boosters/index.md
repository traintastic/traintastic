# Unterstützte Smart-Booster

Diese Seite listet **Smart-Booster**, die von Traintastic unterstützt werden.

Smart-Booster sind Booster, die **Statusrückmeldungen** und/oder **Fernsteuerung** bereitstellen,
zum Beispiel das Ein- oder Ausschalten eines Stromabschnitts über eine Zubehöradresse (Weichenadresse).

Einfache Booster ohne Rückmeldungen oder Steuerfunktionen sind hier nicht aufgeführt, da sie keine zusätzlichen
Funktionen für Traintastic bereitstellen.

## Allgemeine Hinweise

!!! note
    Smart-Booster-Diagnosen sind optional.
    Ein Booster funktioniert auch dann korrekt, wenn einige gemeldete Werte nicht verfügbar oder unzuverlässig sind.

!!! tip "Nicht aufgeführt?"
    Der eigene Smart-Booster kann trotzdem mit Traintastic funktionieren.
    Wenn er mit einem der oben aufgeführten Booster kompatibel ist, kann er ebenfalls funktionieren.
    Erfahrungen bitte im [Community-Forum](https://discourse.traintastic.org) teilen.
