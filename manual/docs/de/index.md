# Willkommen im Traintastic-Handbuch

**Traintastic** ist eine Software zur Steuerung und Automatisierung von Modelleisenbahnen.  

Damit lassen sich Lokomotiven, Weichen, Signale und Zubehör direkt vom Computer aus steuern, Rückmelder abfragen und komplette Anlagen automatisieren.

Egal, ob du gerade deine erste digitale Modellbahn aufbaust oder eine große automatisierte Anlage betreibst – Traintastic hilft dabei, Züge sicher und reibungslos zu betreiben.

## Hauptfunktionen
- Steuerung von Lokomotiven und Zubehör
- Unterstützung mehrerer Digitalsteuerungen
- Echtzeitüberwachung der Anlage
- Multi-User- und Netzwerksteuerung
- Erweiterbar durch Skripting

## So funktioniert Traintastic
Traintastic verwendet eine **Client/Server-Architektur**:

- Der **Server** steuert die Anlage und verwaltet die Automatisierung.
- Der **Client** ist die grafische Oberfläche zur Bedienung und Überwachung der Anlage.

Beides kann auf demselben Computer laufen, oder der Server läuft z. B. auf einem Raspberry Pi, während sich Clients von anderen PCs im Netzwerk verbinden.
