# YaMoRC YD7403

The **YaMoRC YD7403** is a smart booster with extensive diagnostic capabilities.

!!! tip ""
    [**Drenth Design & Consulting B.V.**](http://yamorc.de) kindly donated a **YaMoRC YD7403** to the Traintastic project.
    This contribution made full implementation and testing possible. Thank you!

## Supported diagnostics

- Load
- Temperature
- Current
- Track voltage
- Input voltage
- Short circuit

## Control capabilities

- Power district can be enabled/disabled
- Track output polarity can be switched (normal / inverted)

## Requirements and notes

- YD7403 firmware version **X.Y.Z or newer** is required
- All reported values are stable and reliable

## Configuration

The following configuration options are available:

- **Interface** — The LocoNet interface used for communication with the booster
- **Address** — The LocoNet booster address (factory default: **1**)
